## taichi_smallpt
A small path tracer following the style of [smallpt](https://www.kevinbeason.com/smallpt/) using taichi programming language

### Prerequisites
```
pip install taichi
pip install taichi_glsl
```

### Run
```
python taichi_smallpt.py
```

### Result
* 1000 spp within 20 mins
